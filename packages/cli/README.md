# dns-synchub-cli

CLI package for running `dns-synchub`.

## Entry Points

- `dns-synchub`

## Key Commands

- `dns-synchub --version`
- `dns-synchub --show-config`

## Notes

- Runtime settings are resolved from environment variables and optional `.env` files.
- `--show-config` prints resolved configuration and exits without starting pollers.
